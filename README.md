# Netflix Data Visualization Project   
This project uses advanced visualizations, including bump charts and network graphs, to analyze Netflix data trends. It uncovers ranking shifts, collaboration patterns, and viewer preferences to enhance content recommendation strategies.  

# Data Source 
The dataset used for Netflix Insights is available on Kaggle. You can find the dataset and additional information: https://www.kaggle.com/datasets/shivamb/netflix-shows

## Features  
- Track ranking shifts in Netflix content using bump charts.  
- Visualize collaboration networks between creators and genres.  
- Identify viewer trends to inform personalized recommendations.  

## Tools  
- Python, Pandas, Matplotlib, Seaborn, Plotly, NetworkX.  

## Outcomes  
Improved understanding of content trends, viewer preferences, and collaboration patterns to refine content strategies.
